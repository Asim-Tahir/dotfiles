6. ### Copy files to your home directory

    ```zsh
    cp -a . $HOME
    # or
    cp -a . ~
    ```

7. ### Refresh `.zshrc`

    ```zsh
    source $HOME/.zshrc
    # or
    source ~/.zshrc
    ```
